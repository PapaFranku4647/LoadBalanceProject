Detailed Description

Manages a queue of web requests and a pool of web servers.

The LoadBalancer distributes incoming requests from a central queue to a dynamic pool of WebServer instances. It also manages the simulation clock and dynamically scales the number of servers up or down based on load.

void LoadBalancer::runSimulation ( int  totalTime)

Runs the entire load balancing simulation.

Runs the main simulation loop for a specified duration.

Parameters
totalTimeThe total number of clock cycles to run the simulation for.

This method iterates from clock cycle 0 to totalTime. In each cycle, it:

  1. Potentially adds new random requests to the queue.
  2. Scales the number of web servers up or down based on queue size.
  3. Assigns requests from the queue to idle servers.
  4. Tells each server to process its current request for one cycle.
  5. Prints a status update every 100 cycles.
    Parameters
    totalTimeThe total number of clock cycles for the simulation.
