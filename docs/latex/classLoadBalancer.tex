\doxysubsection{Detailed Description}
Manages a queue of web requests and a pool of web servers. 

The \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}} distributes incoming requests from a central queue to a dynamic pool of \mbox{\hyperlink{classWebServer}{Web\+Server}} instances. It also manages the simulation clock and dynamically scales the number of servers up or down based on load.
