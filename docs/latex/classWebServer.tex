\doxysubsection{Detailed Description}
Simulates a single web server that can process one request at a time. 

A \mbox{\hyperlink{classWebServer}{Web\+Server}} instance can be either idle or busy. When busy, it processes a single request, counting down the required clock cycles until completion.

Simulates one clock cycle of processing for the current request. 

Processes the current request for one clock cycle.

If the server is busy, it decrements the remaining time. When the time reaches zero, it signifies that the request is finished.

If the server is busy, {\ttfamily time\+Remaining} is decremented. If {\ttfamily time\+Remaining} becomes 0, a message is printed indicating the request is finished. \mbox{\Hypertarget{classWebServer_aa69e1165dac0ddd33124cabc4c3326c5}\label{classWebServer_aa69e1165dac0ddd33124cabc4c3326c5}}
